# Sentence-Transformers and all-MiniLM-L6-v2

Sentence-Transformers is a library that provides pretrained models for sentence and document embeddings. The model `all-MiniLM-L6-v2` is a popular choice for RAG systems.

Properties:
- Dimension: 384
- Lightweight and fast on CPU
- Good performance on semantic similarity tasks

It is a strong default for building semantic search and retrieval pipelines without requiring GPUs.
